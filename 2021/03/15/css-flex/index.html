flex弹性布局

前端 css flex 布局
前端 css 布局
发布日期:   2021-03-15
文章字数:   583

弹性布局

1.父级节点

.parentNode {
  display: flex;
}
/* 
如果节点设置了此css属性，相当于在它的所有子节点上添加了浮动属性；
如果父级节点使用了弹性布局，那么它的子结点就不需要添加float属性
*/

2.父节点的其他属性

（1）justify-content
.parentNode {
  display: flex;
  justify-content: center; /* 设置子元素水平居中 */
  justify-content: space-between; /* 设置子元素两端对齐 */
  justify-content: space-around; /* 设置子元素都有独自的左右距离 */
  justify-content: flex-start; /* 设置子元素水平居左，此为默认属性 */
  justify-content: flex-end; /* 设置子元素水平居右 */
}
（2）align-items
.parentNode {
  display: flex;
  align-items: center; /* 设置子元素垂直居中 */
  align-items: flex-start; /* 设置子元素垂直居上 */
  align-items: flex-end; /* 设置子元素垂直居下 */
}
（3）align-content
/* 此属性只对多行子元素生效 */
.parentNode {
  display: flex;
  align-content: center; /* 设置子元素垂直居中 */
  align-content: flex-start; /* 设置子元素垂直居上 */
  align-content: flex-end; /* 设置子元素垂直居下 */
  align-content: stretch; /* 拉伸 ，默认值 */
}
（4）flex-direction
/* 此属性设置子元素的排列方向 */
.parentNode {
  display: flex;
  flex-direction: row; /* 子元素横向排列 */
  flex-direction: row-reverse; /* 子元素横向倒序排列 */
  flex-direction: column; /* 子元素纵向排列 */
  flex-direction: column-reverse; /* 子元素纵向倒序排列 */
}
/* 这里需要注意：当你设置了子元素纵向排列方向后，前两个属性也会随着改变方向！！！ */
（5）flex-wrap
/* 此属性设置子元素是否正常换行 */
.parentNode {
  display: flex;
  flex-wrap: wrap; /* 子元素正常换行 */
  flex-wrap: nowrap; /* 子元素不换行 默认值 */
}
/* 这里需要注意：当你设置了子元素纵向排列方向后，前两个属性也会随着改变方向！！！ */
（6）flex-flow
/* 设置子元素纵向排列正常换行 */
.parentNode {
  display: flex;
  flex-flow: column warp;
}
/* 相当于（4）和（5）的组合 */

3.子元素上的属性

.children {
  flex: 1; /* 用来设置每个节点的宽度，子元素在划分父级宽度的时候，会先排除固定的宽度 */
  align-self: flex-start; /* 此属性可以覆盖父节点的align-items的垂直排列 */
  flex-grow: 1; /* 设置元素的放大比例，默认值为0 */
  order: 0; /* 规定子元素的排列顺序，数值越小越靠前 默认值为0 */
}
